# Example of description of Start Lists

````
{
    identifier : <Start_List_Item_1>,
    bib identifier : '1006',
    score points: 234,   // the points accumulated in the league at start of this race)
    order : 45           // order in the start list
    // transponder identifier  N/A
    // lane N/A
    // features   N/A
}
````
